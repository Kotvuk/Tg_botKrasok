# Telegram-бот для centr-krasok.kz

## Что умеет бот
- 📦 Каталог товаров с фото, ценами, характеристиками
- 🔍 Поиск по названию и бренду
- 🤖 ИИ-консультант (Groq) — только по теме красок
- 🚫 Блокировка посторонних тем

---

## ВАРИАНТ 1: Запуск на своём компьютере (ноутбук должен быть включён)

```
Двойной клик по файлу:  запустить_бота.bat
```

---

## ВАРИАНТ 2: Запуск на Railway (бот работает 24/7, ноутбук можно выключить)

### Шаг 1 — Установить Git
Скачать: https://git-scm.com/download/win  
Установить с настройками по умолчанию.

### Шаг 2 — Зарегистрироваться на GitHub
Сайт: https://github.com  
Создать бесплатный аккаунт.

### Шаг 3 — Создать репозиторий на GitHub
1. Зайти на https://github.com/new
2. Название: `centr-krasok-bot`
3. Нажать **Create repository**
4. Скопировать ссылку репозитория (например: `https://github.com/ВАШ_НИК/centr-krasok-bot.git`)

### Шаг 4 — Загрузить код на GitHub
Открыть командную строку в папке бота и выполнить:
```
git init
git add .
git commit -m "init"
git branch -M main
git remote add origin https://github.com/ВАШ_НИК/centr-krasok-bot.git
git push -u origin main
```
> ВАЖНО: файл .env НЕ загружается (он в .gitignore) — токены вводятся на Railway

### Шаг 5 — Зарегистрироваться на Railway
Сайт: https://railway.app  
Войти через GitHub аккаунт.

### Шаг 6 — Создать проект на Railway
1. Нажать **New Project**
2. Выбрать **Deploy from GitHub repo**
3. Выбрать репозиторий `centr-krasok-bot`
4. Нажать **Deploy Now**

### Шаг 7 — Добавить переменные окружения на Railway
1. Открыть проект → вкладка **Variables**
2. Добавить:
   - `BOT_TOKEN` = ваш токен от @BotFather
   - `GROQ_API_KEY` = ваш ключ от console.groq.com
3. Railway автоматически перезапустит бота

### Готово! 🎉
Бот будет работать 24/7 без вашего ноутбука.  
Railway даёт $5 кредитов/месяц бесплатно — этого хватит на 3-7+ дней непрерывной работы.

## Файлы проекта

| Файл | Описание |
|------|----------|
| `bot.py` | Главный файл — хендлеры Telegram |
| `scraper.py` | Парсинг centr-krasok.kz |
| `ai_assistant.py` | ИИ-консультант (Groq API) |
| `keyboards.py` | Inline-клавиатуры |
| `config.py` | Настройки |
| `Procfile` | Команда запуска для Railway |
| `.env` | Токены (не загружать на GitHub!) |
